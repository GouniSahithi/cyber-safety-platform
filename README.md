# Cyber Safety Platform 🔐

## 📌 Project Overview

The Cyber Safety Platform is a web-based application designed to educate users about online safety, phishing, cyberbullying, and digital threats through interactive modules, quizzes, and personalized recommendations.

---

## 🚀 Features

* 🔑 User Authentication (Login/Register)
* 👤 Profile Management
* 🧠 Interactive Quizzes
* 📊 Performance Tracking
* 🎯 Personalized Recommendations
* 📚 Multi-language Flashcards
* 🎥 Quick Learning Videos

---

## 🛠️ Tech Stack

### Backend:

* Java
* Spring Boot
* Maven

### Frontend:

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

* `backend/` → Spring Boot APIs and logic
* `frontend/` → UI pages and scripts

---

## ⚙️ How to Run

### Backend:

```bash
cd backend/cybersafety
mvn spring-boot:run
```

### Frontend:

Open `index.html` in browser

---

## 🎯 Purpose

This project aims to spread awareness about cyber threats and help users stay safe in the digital world.
