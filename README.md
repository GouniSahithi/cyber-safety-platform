# 🛡️ Cyber Safety Learning Platform

An interactive web platform designed to educate users about cyber safety, online threats, and secure digital practices through videos, quizzes, and real-life scenarios.

---

## 📌 Project Overview

This project focuses on creating awareness about common cyber threats such as phishing, online scams, identity theft, and cyberbullying. It provides a structured learning experience with different difficulty levels and multilingual support.

Users can:

* Learn through videos and explanations
* Test their knowledge using quizzes
* Track their progress
* Generate certificates after completion

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 🌐 Multi-language Support (English, Hindi, Telugu)
* 🎥 Quick Learning Videos
* 📚 Detailed Explanations & Real-life Scenarios
* 🧠 Quiz-Based Evaluation System
* 📊 Progress Tracking
* 🏆 Certificate Generation
* 📱 Responsive UI Design

---

## 🧩 Modules Covered

* Phishing Awareness
* Online Scams
* Identity Threats
* Cyberbullying

Each module includes:

* Easy, Medium, Hard levels
* Tutorials, News, and Scenario-based learning

## ⚙️ How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/GouniSahithi/cyber-safety-platform.git
cd cyber-safety-platform
```

### 2️⃣ Run Backend (Spring Boot)

```bash
cd project/backend/cybersafety
mvn spring-boot:run
```

### 3️⃣ Run Frontend

Open `index.html` in your browser
OR use Live Server (recommended)
